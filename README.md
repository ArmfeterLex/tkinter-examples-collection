# Tkinter Examples Collection

Коллекция простых примеров на Python с графическим интерфейсом tkinter для изучения основ создания GUI-приложений.

## Примеры

### 1️⃣ Сортировка строк
Программа принимает строку с пробелами, разбивает на слова, сортирует их по алфавиту и выводит результат. Демонстрация bind для обработки событий.

### 2️⃣ Класс для блоков
Два блока с полем ввода и кнопкой, использующие один класс:
- Первый блок сортирует слова по алфавиту
- Второй блок переворачивает порядок слов

### 3️⃣ Калькулятор (незавершённый)
Пример структуры калькулятора с радиокнопками для выбора операций. Требует доработки.

### 4️⃣ Изменение свойств кнопки
При нажатии кнопка меняет:
- Текст
- Цвет фона
- Цвет текста
- Цвет при наведении

### 5️⃣ Настройка шрифтов и цветов
Демонстрация различных шрифтов:
- Arial 32
- Comic Sans MS 24 bold
- Настройка границ и фона

### 5️⃣ Выдача
Простое окно с кнопкой "Взять" и меткой, которая меняет текст на "Выдано".

### 6️⃣ Вставка времени
При нажатии кнопки в поле Entry вставляется текущее время в формате ЧЧ:ММ:СС.

## Куда можно встроить

- В учебные материалы по Python
- В курс по основам tkinter
- Как шпаргалку по часто используемым элементам
- В сборник примеров для начинающих

## Требования

- Python 3.x
- tkinter (входит в стандартную поставку Python)

## Примечание

В коллекции есть как завершённые примеры, так и наброски (калькулятор) для самостоятельной доработки. Особенно полезен пример с классом Block — показывает, как создавать повторно используемые компоненты.
